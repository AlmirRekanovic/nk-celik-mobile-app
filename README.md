# NK Čelik Mobile App

Mobile application for NK Čelik football club members with voting, news, and shop features.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Member authentication with Supabase
- Live polls and voting system
- News feed from WordPress
- Admin panel for poll management
- Auto-refresh on screen focus

## Building APK/IPA

Want to test on real devices? See **[BUILD.md](./BUILD.md)** for detailed instructions on:
- Building Android APK files
- Building iOS IPA files
- Using Expo Go for quick testing
- Distributing to friends via TestFlight

## Admin Access

- First Name: `admin`
- Last Name: `admin`

## Tech Stack

- React Native with Expo
- Expo Router for navigation
- Supabase for database and authentication
- WordPress REST API for news
- TypeScript
